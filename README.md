# VoiceWeave

VoiceWeave is a small React app for creating and joining lightweight "weaves" and polls — collaborative sessions that let users vote and visualize results in real time.

Live demo: https://voiceweave-hv.web.app/

## Features
- Create and join weaves
- Create polls and vote
- Visualize poll results

## Quick start
1. Install dependencies:

   npm install

2. Start the dev server:

   npm start

3. Build for production:

   npm run build

## Tech
- React
- Firebase (Auth, Firestore, Hosting)
